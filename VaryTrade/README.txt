com.VaryTrade.Controller contains all controller classes for web applicaion and REST API.
com.VaryTrade.DAO contains data access actions.
com.VaryTrade.Model contains all model classes.
Model classes prefixed with Web are all form backing objects.
com.VaryTrade.Service contains all service actions such as producing reCAPTCHAs, sending emails, and creating Hyperwallet entities.
com.VaryTrade.Validator contains all classes used to validate user form input.
